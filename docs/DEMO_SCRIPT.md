# SupportSignal — Demo Script (3–5 min)

Audience: recruiter / hiring manager for analytics engineering, data product, full-stack analytical roles.

## Setup

Open **https://supportsignal-lab-lake.vercel.app** (or local `npm run dev` in `apps/web`).

## Script

| Min | Action | What to say |
|---:|---|---|
| 0:00 | Point to lab banner | “This is a **lab**, not a helpdesk. Heuristics + human review — I am not claiming autonomous AI support.” |
| 0:30 | Click **Reexecutar demo one-click** | “Synthetic inbox, 240 messages. Classification runs in the browser for a stable public demo.” |
| 1:00 | Inbox + SLA KPIs | “We measure first-response SLA and breach rate — operational signal, not vanity charts.” |
| 1:40 | Topic classifier | “Configurable keyword rules. Explainable by design; LLM can come later with review gates.” |
| 2:20 | Refund risk board | “Score ≥ 70 is a **watchlist**. Drivers are listed — risk is heuristic, not a calibrated ML probability.” |
| 3:00 | Root-cause + memo | “Symptoms become cause hypotheses and a weekly memo with P0–P2 actions.” |
| 3:45 | Claims panel | “Explicit non-claims: no auto-replies, no production mailbox, no enterprise magic.” |
| 4:30 | Close | “Portfolio value: pragmatic text classification, operational metrics, responsible product framing.” |

## Do not say

- “AI that resolves support”
- “Production-ready helpdesk”
- “Enterprise / premium ML refund prediction”
