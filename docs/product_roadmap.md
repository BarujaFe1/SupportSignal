# SupportSignal — Product Roadmap

## MVP (shipped in this portfolio release)
- CSV / demo inbox import
- Topic classification (rule-based, configurable)
- Sentiment / urgency heuristics
- SLA first-response metrics and breach flags
- Refund risk board with explainable drivers
- Root-cause explorer
- Weekly support memo + action backlog
- Billing-ready positioning (tiers documented; Stripe/MP later)

## Phase 2
- Gmail / Zendesk / Intercom / WhatsApp provider connectors
- Spike alerts
- Suggested replies (human-approved)
- Lightweight knowledge base links
- Churn / refund cohort analysis

## Phase 3
- Product improvement workflow
- Automatic routing hints
- Near-real-time monitoring
- Category benchmarking

## Explicit non-goals (early)
- Full helpdesk replacement
- Fully autonomous customer replies
- Blind LLM automation without human review
